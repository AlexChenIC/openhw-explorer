# Programs

> programs 是 OpenHW 全部项目的项目级/治理文档仓库：TWG 与各 Task Group 会议纪要、每个项目的 gate 材料、项目 dashboard 与流程模板。

数据核对日期: 2026-07-16

## Public summary

programs is OpenHW's governance and project-history repository. It collects Technical Working Group and Task Group records, project-gate packages, dashboard data, process templates, and RTL Freeze material, allowing readers to trace how a project moved from concept through launch, plan approval, and freeze. Contributors and researchers should use it when they need first-hand evidence about project status, decisions, or governance rather than technical RTL or user documentation.

## 中文介绍

programs 是 OpenHW 的项目治理与历史资料仓库。它汇集 Technical Working Group 和 Task Group 记录、项目 gate 材料、dashboard 数据、流程模板与 RTL Freeze 文档，使读者能够追踪项目从概念、立项、计划批准到冻结的过程。贡献者和研究人员需要核对项目状态、决策或治理依据时应查看这里；它并不是 RTL 或用户手册仓库。

## Key facts

- Scope: program- and project-level documentation for OpenHW projects
- Governance records: Technical Working Group plus cores, hardware, and verification Task Group minutes, reports, and presentations
- Project gates: Project Concept, Project Launch, Plan Approve, and Project Freeze material
- Dashboard and process: public project dashboard, gate templates, and RTL Freeze templates
- Boundary: software Task Group material is tracked separately in core-v-sw

## 项目概述

programs 仓库 README 列出的目录：TWG（Technical Working Group 纪要与演示材料）、TG（cores/hw/verification 三个 Task Group 的月报与纪要，sw task group 在 core-v-sw 仓库跟踪）、OpenHW-Project-Descriptions-and-Plans（每个 OpenHW 项目一个目录，存放四个 gate 的材料，可一览所有项目）、OpenHW-dashboard（官网链接的项目仪表盘 markdown）、process（流程与模板文档）、Attendance-tracking（会议出席记录）。
